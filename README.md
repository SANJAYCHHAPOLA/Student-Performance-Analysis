# Student Performance Analysis

## Project Description
This project focuses on analyzing student academic data using data analytics techniques. 
The system studies factors such as attendance, study hours, internal marks, and previous academic performance to predict whether a student will pass or fail. 
The project helps in identifying students who are at academic risk and supports early intervention to improve performance.
